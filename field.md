# Предлагаемые технические поля для редукторов

1. **Тип редуктора** – выпадающий список.  
   *Тип данных:* `select` (значения: `цилиндрический`, `червячный`, `планетарный`, `конический`, `коническо-цилиндрический`, `комбинированный`).  
   *Режим ввода:* возможна опция «Другое» с ручным заполнением.  
   *Применение:* влияет на подсказки по КПД, диапазонам U и выбору подшипников.

2. **Количество ступеней** – целое число (1–6).  
   *Тип данных:* `integer` + единица измерения «ступ.»  
   *Заполнение:* выпадающий список 1–6, при выборе «другое» разрешить ручной ввод.  
   *Валидация:* >0, для комбинированных допускается 6.

3. **Номинальный крутящий момент** – числовое поле, единицы «Н·м».  
   *Тип данных:* `number`, шаг 10/100 в зависимости от диапазона.  
   *Валидация:* >0, опционально контроль отношения момент/масса (для цилиндрических <30 Н·м/кг — предупреждение).  
   *Заполнение:* свободный ввод, но с подсказками (например, 500, 800, 1600…).

4. **Передаточное отношение (U)** – числовое поле без единицы (размерность «—»).  
   *Тип данных:* `number` (дробное), шаг 0.1.  
   *Валидация:* >0.5 и <1000; при выходе за типовой диапазон отображать предупреждение.  
   *Заполнение:* предлагаемые значения из справочника типоразмеров, но разрешён ручной ввод.

5. **Материал корпуса** – список (`Чугун`, `Высокопрочный чугун`, `Сварной стальной`, `Стальной литой`, `Алюминиевый сплав`, `Нержавеющая сталь`, `Комбинированный`).  
   *Тип данных:* `select` + опция «Другое (ручной ввод)».  
   *Единицы:* не требуются.  
   *Примечание:* при выборе «комбинированный» стоит собрать уточнение (например, «чугун + алюминий»).

6. **Устанавливаемые подшипники** – множественное поле.  
   *Тип данных:* `multi-select + autocomplete`.  
   *Формат:* `производитель / серия` (например, `SKF 32210`, `C&U 3205`, `ГОСТ 180210`).  
   *Заполнение:* список рекомендуемых брендов (`SKF`, `FAG`, `Timken`, `РФ ГОСТ`, `Высокоточный Китай`) + свободный ввод.

## Формат хранения

- Отдельный блок «Технические параметры редуктора» в `formData` (один документ = один набор параметров).
- Поля добавляются под таблицей позиций во вкладке «Техническое приложение».
- Набор хранится как объект `reducerSpecs`:

  ```json
  {
    "type": "цилиндрический",
    "customType": "",
    "stages": 3,
    "torqueNm": 1600,
    "ratio": 25.4,
    "housingMaterial": "Чугун",
    "housingMaterialNote": "",
    "gearMaterial": "Сталь 18CrNiMo7-6 цементация, шлифовка зуба",
    "gearMaterialNote": "",
    "additionalInfo": "",
    "bearings": ["SKF 32210", "SKF 3209"],
    "pdfFields": {
      "type": true,
      "stages": true,
      "torqueNm": true,
      "ratio": true,
      "housingMaterial": true,
      "gearMaterial": true,
      "bearings": true,
      "additionalInfo": true
    }
  }
  ```

- Каждое поле поддерживает режим «из списка» + ручной ввод (если нужного значения нет).
- Для каждого параметра можно включать/выключать вывод в PDF — состояние сохраняется в блоке `pdfFields`.

## Рекомендации по остальным параметрам

- **КПД и тепловая мощность**: для цилиндрических/конических – давать диапазон КПД (например, 0,92–0,97), для червячных – 0,7–0,85. Можно хранить как «расчётный КПД» + «условия охлаждения».
- **Способ монтажа**: фланцевый, лаповый, вертикальный, с реактивным рычагом.
- **Направление вращения**: реверсивный, нереверсивный.
- **Условия эксплуатации**: температура (мин/макс), влажность, категория по ГОСТ 15150 (У, УХЛ, Т). Позволит автоматически подбирать покрытие/уплотнения.
- **Тип Смазки**: тип масла (ISO VG 220 и т.п.), объём, интервал обслуживания.
- **Система смазки**: картерный окунанием, принудительный встроенный насос, принудительный от внешней маслостанции.
- **Масса**: вводить вручную.
- **габариты**: вводить вручную.
- **Стандарты/сертификаты**: ГОСТ, ISO, ATEX и т.п.; можно сделать чекбоксы + поле «другие требования».
- **Доп. опции**: датчики вибрации/температуры, системы охлаждения, специальные покрытия (например, для пищевой промышленности).

Все поля позволяют ручной ввод иных данных.
